

# Clean&Care

Clean&Care is an API for a  car clean&care company mobile app.

## Installation & Run

#### Download project
```
 git clone git@github.com:handesahin/clean-and-care.git
 ```
 
 #### Database migration & seed
```
 php artisan migrate --seed
 ```
 
  #### Load car details
```
 php artisan cars:upsert
 
  ```
 #### Run project
```
 php artisan serve
 
  ```
