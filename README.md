

#Clean&Care

Clean&Care is an api projects for a  car clean&care company mobile app.

##Installation & Run

> ####### Download project
>
> _git clone git@github.com:handesahin/clean-and-care.git_
> 

> ####### Database migration & seed
>
> _php artisan migrate --seed_
>  
> 

> ####### Load car details
>
> _php artisan cars:upsert_


>  ####### Run project
>
> _php artisan serve_
